# 开发计划与进度追踪（develop_plan）

## 1. 文档目的
- 对项目需求进行分阶段落地，统一开发节奏。
- 作为唯一进度追踪文档，后续开发按本计划执行并持续更新。

## 2. 项目范围（基于 PRD）
- 用户管理：增删改查、启用禁用、导入导出、个人中心、重置密码。
- 组织管理：树形组织、无限层级、父子关系、排序、成员关联。
- 角色权限：角色管理、菜单/按钮权限、数据权限规则。
- 系统支撑：登录认证、JWT、验证码、日志审计、安全加固。

## 3. 技术基线
- 后端：Spring Boot 3 + JDK 21 + PostgreSQL + MyBatis + MyBatis-Plus。
- 前端：Vue 3。
- 数据库：`user-management`（UTF8）。
