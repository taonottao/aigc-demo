# 开发计划与进度追踪（develop_plan）

## 1. 文档目的
- 基于当前代码现状维护阶段计划与执行节奏。
- 作为迭代看板，统一记录里程碑、待办与阻塞项。

## 2. 项目范围（基于 PRD）
- 用户管理：增删改查、启用禁用、导入导出、重置密码、角色绑定。
- 组织管理：树形组织、父子关系、成员关联、排序管理。
- 角色权限：角色管理、菜单/按钮权限、数据权限规则。
- 系统支撑：登录认证、JWT、验证码、日志审计、安全策略。

## 3. 技术基线
- 后端：Spring Boot 3 + JDK 21 + PostgreSQL + MyBatis + MyBatis-Plus。
- 前端：Vue 3。
- 数据库：`user-management`（UTF8）。
