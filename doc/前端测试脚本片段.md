
# 前端测试脚本片段（Playwright）

> 说明：本文件提供“可直接照着补全”的 UI 自动化脚本片段（以 Vite dev 为例）。
> - 前端：`http://localhost:5173`
> - 后端：`http://localhost:8080`（Vite 已将 `/api` 代理到 8080）

## 0. 登录流程（captcha + JWT）
```ts
import { test, expect } from '@playwright/test'

test('login', async ({ page }) => {
  await page.goto('http://localhost:5173/login')

  // 1) 刷新验证码（按钮显示“刷新：XXXX”）
  await page.getByRole('button', { name: /刷新：/ }).click()
  const captchaLabel = await page.getByRole('button', { name: /刷新：/ }).textContent()
  const captcha = (captchaLabel || '').replace('刷新：', '').trim()
  await expect(page.getByRole('button', { name: /刷新：/ })).toBeVisible()

  // 2) 输入账号密码 + 填写验证码（默认账号：smile-admin / 123456）
  await page.getByLabel('用户名').fill('smile-admin')
  await page.getByLabel('密码').fill('123456')
  await page.getByLabel('验证码').fill(captcha)

  // 3) 登录
  await page.getByRole('button', { name: '登录并进入系统' }).click()
  await expect(page).toHaveURL(/\/dashboard$/)
})
```
